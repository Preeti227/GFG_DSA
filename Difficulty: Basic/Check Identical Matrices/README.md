<h2><a href="https://www.geeksforgeeks.org/problems/identical-matrices1042/1">Check Identical Matrices</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given two square matrices <strong>mat1[][] </strong>and <strong>mat2[][]</strong> of size <strong>n × n</strong>, determine whether the matrices are identical or not.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>mat1[][] = [[1, 2], [3, 4]], mat2[][] = [[1, 2], [3, 4]]
<strong>Output: </strong>true
<strong>Explanation: </strong>Both the matrices are identical, so the answer is 1.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>mat1[][] = [[1, 2], [3, 4], mat2[][] = [[1, 2], [3, 2]]
<strong>Output: </strong>false
<strong>Explanation: </strong>Both the matrices are not identical, So, answer is 0.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong><br>1 ≤ n ≤ 1000<br>-1000 ≤ mat1[i][j], mat2[i][j] ≤ 1000</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Matrix</code>&nbsp;
